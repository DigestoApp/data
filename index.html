Digesto API V2 Documentation

This document provides detailed information on how to interact with the Digesto API to retrieve data. The API follows RESTful principles and returns data in JSON format.

Endpoint Structure

The API's endpoint URL structure is as follows:

https://digesto-content-public.s3.us-east-2.amazonaws.com/{year}/{month}/{day}/{edition}.json
  • {year} - 4-digit year (yyyy).
  • {month} - 2-digit month (MM).
  • {day} - 2-digit day (dd).
  • {edition} - Edition of the day. Possible values include:
    • morning - Available at 8AM EST and later.
    • afternoon - Available at 2PM EST and later.
    • evening - Available at 7PM EST and later.

API Response Format

{
  "stories": [
    {
      "title": "string",
      "summary": "string",
      "location": "string",
      "category": "string",
      "info": "string"
    }
  ]
}

Example Usage

Below are example API calls:

https://digesto-content-public.s3.us-east-2.amazonaws.com/2024/02/15/morning.json
https://digesto-content-public.s3.us-east-2.amazonaws.com/2024/03/04/evening.json
Note: Ensure your API requests are correctly formatted as per the guidelines. For support, open an issue on GitHub at DigestoApp/Data.
